# Ruboty::Maekawa
みくだにゃ

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'ruboty-maekawa', 
    :git => "git@github.com:mopp/ruboty-maekawa.git",
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-maekawa
